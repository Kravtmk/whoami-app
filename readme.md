# WhoAmI — Personal Time Intelligence App

## Overview
WhoAmI is a self-reflection and time tracking backend application.

The system allows structured daily logging of time allocation 
based on roles (DevOps, Family, Health, etc.).

## Tech Stack
- FastAPI
- Docker
- Docker Compose
- Persistent Volumes
- JSON storage (phase 1)
- Telegram Bot (planned)
- AI layer (planned)

## Architecture
User → API → JSON DB (Docker Volume)

## Features implemented
- Dockerized backend
- Persistent volume storage
- Environment configuration (.env)
- HTML test interface
- CRUD roles
- Daily log tracking

## Roadmap
